This Power BI dashboard provides a comprehensive analysis of sales performance across regions, product categories, and customer segments. It helps identify key trends, top-performing products, and profit insights to support data-driven decision-making.

## Features
- Interactive filters for region, category, and customer segment  
- KPIs for Total Sales, Profit, Quantity Sold, and Average Discount  
- Dynamic charts showing:
  - Sales by Region and State
  - Category-wise Profit Analysis
  - Customer-wise Purchase Trends
  - Monthly Sales Performance

- Power BI Desktop  
- Data Cleaning: Power Query  
- Data Visualization: Power BI Dashboard, DAX Measures
